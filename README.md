# exemplo-else-if

Classificador de Desempenho

Aplicação web simples feita com HTML, CSS e JavaScript puro para classificar o desempenho de um aluno a partir de uma nota informada pelo usuário. O sistema funciona totalmente no navegador e não utiliza banco de dados nem armazenamento permanente.

O usuário digita um valor de 0 a 10 e, ao clicar no botão de análise, o programa executa uma função que interpreta a nota usando estruturas condicionais. O resultado é exibido na própria página com uma mensagem descritiva e mudança de cor para facilitar a leitura.

Regras de Classificação

9 ou mais → Excelente

7 ou 8 → Bom

5 ou 6 → Regular (recuperação)

Abaixo de 5 → Insuficiente

Como funciona

O JavaScript captura o número digitado no campo de entrada, converte para valor numérico e compara com as faixas definidas. Dependendo da condição atendida, o texto do feedback e a cor são atualizados dinamicamente no elemento da página.

Objetivo do projeto

Praticar:

captura de dados do usuário

manipulação do DOM

uso de if / else if / else

respostas visuais dinâmicas
